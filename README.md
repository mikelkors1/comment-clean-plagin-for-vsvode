# Clean C++ Comments

Простое расширение для VS Code для удаления комментариев из C++ кода.

## Что умеет

- Удаляет однострочные комментарии `//`
- Удаляет многострочные комментарии `/* */`
- Удаляет все комментарии сразу
- Сохраняет исходную структуру кода

## Как использовать

1. Откройте C++ файл (.cpp, .h, .hpp)
2. Нажмите `Ctrl+Shift+P`
3. Введите `Clean C++ Comments`
4. Выберите тип удаления:

**Доступные команды:**
- **Удалить // комментарии** - удаляет только `//` комментарии
- **Удалить /* ... */ комментарии** - удаляет только `/* */` комментарии  
- **Удалить все комментарии** - удаляет все типы комментариев

## Пример

**Было:**
```cpp
#include <iostream>
// комментарий
int main() {
    return 0;
}
```

**Стало**
```cpp
#include <iostream>

int main() {
    return 0;
}
```
